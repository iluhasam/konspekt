# Конспект — контекст проекта

## Что это

Телеграм-бот, который делает **конспект** из статьи, YouTube-видео или
документа и присылает его сообщениями прямо в чат. Личный инструмент против
«цифрового хлама» — сохранёнок, которые никогда не открываются.

## Зачем

Боль: «сохранил на потом и не открыл», длинные видео и статьи, которые жаль
смотреть/читать целиком. Решение: мгновенная выжимка сути (суть + тезисы + теги)
в Telegram и лёгкий триаж кнопками. Контент не копится в кладбище, а сразу
превращается в короткий читаемый конспект.

## Ключевые инварианты

- **Точка входа — только Telegram-бот.** Веба нет.
- **Выход — сообщения в Telegram.** Весь конспект бот шлёт в чат.
- **Obsidian — только про проект.** Сюда идёт документация (контекст, ADR,
  Dev Log), а НЕ сохранённый пользователем контент.
- **Секреты только в `.env`.** Ключ OpenRouter и токен бота в git не попадают.
- **Три типа входа в MVP:** статья (URL), YouTube (транскрипт), документ
  (PDF/DOCX). Остальное — честный отказ.

## Стек

TypeScript · grammY (long-polling) · Prisma + SQLite · OpenRouter (LLM,
модель через env) · `@extractus/article-extractor` · `youtube-transcript` ·
`pdf-parse` · `mammoth`.
